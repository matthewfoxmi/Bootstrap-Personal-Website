Fitness Center: Click to See Project

This is a group project that I worked on, it is a console application that allows the user to join a fitness center, quit their membership, check their membership status, and exit from the program. The user has the option to join a single fitness center, or to join with a multi-center membership, which will display membership points if they select to view their information; the user also has a membership fee depending on what plan they have. This project demonstrates my understanding of the relationship between classes, as well as using conditionals, loops, and methods to call upon.

Dice Roller: Click to See Project

This is a console application I made that allows the user to roll a pair of dice with the amount of sides that they choose. It also has a method built in for when the user chooses 6 sided dice: it will display custom messages depending on what they roll. This project highlights the use of conditionals, methods, and looping.
